## 新增需求

### 需求：AppLayout 主布局
系统**必须**提供包含毛玻璃导航栏、固定侧边栏（桌面端）、底部 Tab 栏（移动端）的响应式主布局。

#### 场景：桌面端布局
- **当** 屏幕宽度 ≥ 1024px（lg）
- **那么** 显示固定顶部导航栏（72px）+ 左侧固定侧边栏（260px）+ 右侧主内容区
- **并且** 侧边栏底部显示"今日目标"进度卡片

#### 场景：移动端布局
- **当** 屏幕宽度 < 1024px
- **那么** 显示精简顶部导航栏（64px）+ 底部 Tab 栏（5 项）
- **并且** 侧边栏以抽屉形式从左侧滑出（点击汉堡菜单）

### 需求：毛玻璃导航栏（Navbar）
导航栏**必须**固定在顶部，使用毛玻璃效果（backdrop-filter blur 16px），包含 Logo、导航链接、主题切换、通知、用户头像。

#### 场景：导航栏内容
- **当** 页面加载
- **那么** 导航栏显示：左侧 Logo（渐变图标 + "学英文"文字）、中部导航链接（首页/课程/考试/口语/统计）、右侧（主题切换/通知/头像）

#### 场景：导航栏响应式
- **当** 屏幕为移动端
- **那么** 中部导航链接隐藏，右侧仅保留关键操作

### 需求：侧边栏（Sidebar）
桌面端固定侧边栏**必须**包含用户信息区、导航菜单（带图标和徽标数）、管理菜单分组、今日目标进度卡片。

#### 场景：侧边栏菜单项
- **当** 渲染侧边栏菜单
- **那么** 显示以下项：学习首页、单词学习（带数字徽标）、听力训练、智能考试、口语评测、翻译查词、分隔线、我的课程、学习统计、设置

#### 场景：活动菜单高亮
- **当** 当前路由匹配某个菜单项
- **那么** 该菜单项显示蓝色背景 + 蓝色文字 + 左侧指示条

### 需求：移动端底部 Tab 栏
移动端**必须**显示固定底部导航栏（lg 以下可见），包含 5 个 Tab 项，中间“学习”按钮突出显示为渐变圆形。

#### 场景：底部 Tab 渲染
- **当** 屏幕为移动端
- **那么** 底部固定显示：首页、课程、学习（突出渐变圆形，略高于其他 Tab）、口语、我的

#### 场景：安全区域
- **当** 设备有底部安全区（如 iPhone）
- **那么** Tab 栏自动适配 `env(safe-area-inset-bottom)`

**交叉引用**：依赖 `frontend-theme`（主题变量驱动颜色）、`frontend-ui-components`（使用 Badge 组件显示通知数）
