## 新增需求

### 需求：考试列表页
系统**必须**提供考试列表页面（`/exams`），展示所有考试记录，支持分页和搜索。

#### 场景：考试列表加载
- **当** 用户进入考试列表页
- **那么** 调用后端获取考试列表（需要新增 JSON API 替代原 PartialView `/exam/examlist`）
- **并且** 以数据表格或卡片网格展示

#### 场景：创建新考试
- **当** 用户点击"创建考试"并设置题目数量和限时
- **那么** POST 请求 `/exam/saveexam?examcount=xxx&limittime=xxx`
- **当** 创建成功
- **那么** 跳转到考试答题页面

#### 场景：删除考试
- **当** 用户确认删除考试
- **那么** POST 请求 `/exam/deleteexam?examid=xxx`

### 需求：考试答题页
系统**必须**提供考试答题页面（`/exams/:id`），按照 DESIGN_SYSTEM.md 定义的考试页面布局设计。

#### 场景：加载考试题目
- **当** 用户进入考试答题页
- **那么** 调用 `GET /exam/getexam?examid=xxx` 获取考试基本信息
- **并且** 调用获取题目内容列表（需要新增 JSON API）

#### 场景：答题交互
- **当** 用户选择选项
- **那么** 选中选项高亮（蓝色边框）
- **并且** 显示当前进度（题目 N/总数）和进度条

#### 场景：答题倒计时
- **当** 考试有时间限制
- **那么** 右上角显示倒计时（⏱️ 格式）

#### 场景：提交答案
- **当** 用户完成所有题目或点击提交
- **那么** POST 请求 `/exam/insertexamnswer?data=xxx&examid=xxx&type=xxx&score=xxx`
- **并且** 跳转到考试结果页

#### 场景：正确/错误反馈
- **当** 提交单题答案（即时模式）
- **那么** 正确答案绿色闪烁 + scale-in 动画；错误答案红色抖动

### 需求：考试结果页
系统**必须**提供考试结果页面（`/exams/:id/result`），展示成绩、正确率、错题详情。

#### 场景：结果展示
- **当** 用户进入考试结果页
- **那么** 显示分数（大号渐变文字）、正确率、答题用时
- **并且** 显示错题列表（每题正确答案和用户答案对比）

#### 场景：成绩庆祝动画
- **当** 考试结果为高分（≥90）
- **那么** 播放烟花庆祝动画（confetti）和分数弹出（bounce-in）

#### 场景：重新考试
- **当** 用户点击"重新考试"
- **那么** POST 请求 `/exam/restartexam?examid=xxx&type=xxx`
- **并且** 跳转到答题页重新开始

**交叉引用**：依赖 `frontend-data-layer`、`frontend-ui-components`（Card/Progress/Table）、`frontend-layout`
