## 新增需求

### 需求：听力练习页
系统**必须**提供听力练习页面（`/hearing/:courseId`），支持音频播放、字幕显示、中英对照。

#### 场景：听力内容加载
- **当** 用户进入听力练习页
- **那么** 调用 `GET /hearing/hearingconent?kc=xxx&id=xxx` 获取字幕数据
- **并且** 返回数据包含每句的 StartTime、EndTime、Value（英文）、ValueCN（中文）

#### 场景：音频播放控制
- **当** 用户点击播放按钮
- **那么** 播放音频，当前正在播放的字幕句子高亮
- **当** 音频播放到某个时间段
- **那么** 自动滚动到对应字幕行

#### 场景：波形动画
- **当** 音频正在播放
- **那么** 播放按钮旁显示波形动画（waveform keyframe）

#### 场景：中英对照模式
- **当** 用户切换显示模式
- **那么** 可选：仅英文、仅中文、中英对照

**交叉引用**：依赖 `frontend-data-layer`、`frontend-ui-components`、`frontend-layout`
