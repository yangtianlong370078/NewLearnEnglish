## 新增需求

### 需求：Button 按钮组件
系统**必须**提供 6 种按钮类型（Primary/Secondary/Ghost/Danger/Gradient/Icon）和 5 种尺寸（xs/sm/md/lg/xl），支持加载态、禁用态和完整的交互动画。

#### 场景：Primary 按钮交互
- **当** 用户悬浮 Primary 按钮
- **那么** 按钮背景加深、出现蓝色阴影、微微上移（-0.5 translate-y）
- **当** 用户点击按钮
- **那么** 按钮回到原位、阴影消失

#### 场景：加载态按钮
- **当** 按钮处于加载状态
- **那么** 显示旋转 spinner 图标 + "加载中..." 文字
- **并且** 按钮禁用，不可点击

### 需求：Card 卡片组件
系统**必须**提供基础卡片、指标卡片（StatCard）、单词卡片（WordCard）三种变体。

#### 场景：卡片悬浮效果
- **当** 用户悬浮卡片
- **那么** 卡片阴影增强、上移 6px（`-translate-y-1.5`），过渡 300ms

#### 场景：单词卡片展示
- **当** 渲染单词卡片
- **那么** 显示单词、音标（等宽字体）、释义（最多两行）
- **并且** 顶部有渐变装饰条（hover 时从左向右展开）
- **并且** 底部有发音按钮和收藏按钮

### 需求：Input 输入框组件
系统**必须**提供带图标的输入框组件，支持搜索、表单等场景，圆角 xl（12px）。

#### 场景：输入框聚焦
- **当** 用户点击输入框
- **那么** 边框变为蓝色，出现 4px 蓝色 focus ring
- **并且** 占位文字颜色为 content-tertiary

### 需求：Modal 模态框组件
系统**必须**提供带遮罩的模态框组件，支持标题栏、内容区、底部操作按钮区域。

#### 场景：模态框打开动画
- **当** 模态框打开
- **那么** 背景渐变模糊、内容区域 scale-in 放大进入（300ms）

#### 场景：模态框关闭
- **当** 用户点击遮罩或关闭按钮
- **那么** 模态框触发关闭回调

### 需求：Badge/Tag 标签组件
系统**必须**提供状态标签（带圆点）和模块标签，支持 6 种预设颜色主题。

#### 场景：状态标签渲染
- **当** 渲染"已完成"状态标签
- **那么** 显示绿色背景 + 绿色圆点 + "已完成"文字

### 需求：Progress 进度条组件
系统**必须**提供线性进度条和环形进度条（SVG），支持渐变填充和过渡动画。

#### 场景：进度条动画
- **当** 进度值更新
- **那么** 进度条宽度平滑过渡（700ms ease-out），使用蓝紫渐变

### 需求：Table 数据表格组件
系统**必须**提供带排序、分页、hover 行高亮的数据表格组件。

#### 场景：表格行悬浮
- **当** 用户悬浮表格行
- **那么** 行背景变为 surface-secondary，平滑过渡

### 需求：Toast 消息提示组件
系统**必须**提供右上角固定位置的 Toast 通知，支持成功/错误/警告/信息四种类型。

#### 场景：成功 Toast
- **当** 操作成功触发 Toast
- **那么** 从右侧滑入显示（400ms），带绿色图标
- **并且** 3000ms 后自动消失

#### 场景：错误 Toast
- **当** 操作失败触发 Toast
- **那么** 显示红色图标
- **并且** 5000ms 后自动消失

### 需求：Skeleton 骨架屏组件
系统**必须**提供可组合的骨架屏组件，用于数据加载过程中的占位展示。

#### 场景：单词列表加载
- **当** 单词列表数据正在请求中
- **那么** 显示单词卡片骨架屏网格，带闪烁（shimmer）动画

### 需求：EmptyState 空状态组件
系统**必须**提供空状态占位组件，包含图标、标题、描述文字和操作按钮。

#### 场景：无学习记录
- **当** 列表数据为空
- **那么** 显示圆形图标容器 + 描述文字 + "开始学习"渐变按钮
